The hackathon will feature projects on the following broad themes:

- Biological simulators (e.g. for population genetics).
- Machine learning/deep learning.
- Web portals for bioinformatic applications, WebAssembly, interactive visualisation.
- Biological data visualisation.
- Parallelisation, optimisation, GPU acceleration for bioinformatic methods.
- Biological imaging.

Project submissions are now closed. Below is a list of the titles and leaders of the projects for the hackathon (in no particular order):

- MAGMA: Multiomics-Augmented Gut Microbiome Agent-based model, *Juan Escorcia, EMBL Heidelberg*
- TailStat, *Jaiganesh Jagadeesh, GIMM*
- self.dna, *Alessandro Lussana, EMBL-EBI*
- MGnify Contig Explorer, *Vikas Gupta, EMBL-EBI*
- Brillouin hyperspectral images storage, visualisation and analysis, *Carlo Bevilacqua, EMBL Heidelberg*
